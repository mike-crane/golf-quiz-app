# Golf Quiz App

A quiz application that tests a users knowledge around the game of golf. After answering 10 multiple choice questions, users are shown their 'Golf IQ'. 

## Demo

- [Live Demo](https://mike-crane.github.io/golf-quiz-app/)

## Built With

* HTML
* CSS
* JavaScript
* jQuery
